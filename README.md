## How to run

Clone the repository:

```
$ git clone https://github.com/Matheuslm10/repo-name.git
```

Enter the directory:

```
$ cd repo-name
```

Install the dependencies:

```
$ yarn install
```

Run the application:

```
$ yarn start
```

Now you can access `http://localhost:3000` in your browser.

### Linting and Format
